## Collection of utilities to make PDP11 analysis in modern world

Removing strange formats, converting from one output to another, converting OCTAL to HEX, or binary to OCTAL.
Used with makefile to build complete debug system.

The only external program to use is "PDP_macro11\macro11"

Other utilities:
DSS = "TOOLS\disas11.exe"
COR = "TOOLS\bin2core.exe"
CLN = "TOOLS\pdp11guilistingcleaner.exe"
OHX = "TOOLS\octal2hex.exe"
L2B = "TOOLS\lst2bin.exe"

are selfmade and available in this GitHub.
